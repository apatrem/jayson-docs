# confidential/ — Trust tier: **confidential**  ⚠️

Your real client work — past proposals and delivered projects. This is the most
useful folder for *quality* (it shows your house style and methodology) and the
most dangerous for *confidentiality*.

Drop here:

- Past **commercial proposals** (won and lost).
- Delivered **project decks, reports, models**.

**Format tip:** the most token-efficient way to reuse this material is an
**anonymised `.md` methodology note** per engagement (structure + approach, no
client name) — write it once and skills read that instead of re-parsing the
heavy original.

**How this folder is used:** deliverable skills may learn **structure, tone, and
methodology** from these — but must **anonymise every client name and confidential
figure**. A past proposal for "ACME Corp" becomes "a European industrial group".

**Hard rule (enforced by every skill):** never output a client name found here
unless that same client is *also* a named public reference in `public/`. When in
doubt, anonymise (sector + geography + year).
