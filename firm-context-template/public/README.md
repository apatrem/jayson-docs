# public/ — Trust tier: **public**

Material you are happy to show the world — the firm's public face.

Drop here:

- **Marketing** decks and one-pagers, company presentations.
- **White papers** and published thought-leadership.
- **Named public references** / case studies you are allowed to cite by name
  (e.g. clients listed on your website).

**Format tip:** add a short `.md` summary alongside any heavy `.pdf`/`.pptx` —
skills read the markdown cheaply and fall back to the original only when needed.

**How this folder is used:** deliverable skills may **quote this freely** and
**name the references** — it is approved public IP. Great grounding for "who we
are", relevant case studies, and sector context.

**Rule of thumb:** if you'd put it on your website or hand it to a prospect
unprompted, it belongs here.
